Quick Start
  1. Start services: docker-compose up --build
  2. Generate a token: node scripts/generate-token.js
  3. Paste the token into the field above and click Save
  4. Go to Students tab — create a student
  5. Go to Exams tab — create an exam
  6. Go to Enrolment tab — select an exam, select a student, click Enrol
