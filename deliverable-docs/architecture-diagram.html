Synchronous Communication (REST / HTTP)

🔗 Cross-Service Calls via axios

Exam Service ⟶ Student Service GET /api/students/:id — verify student exists before creating an exam entry
Transcript Service ⟶ Student Service GET /api/students/:id — pull student profile for transcript
Transcript Service ⟶ Result Service GET /api/results/student/:id — pull semester results & GPA for transcript
Asynchronous Event Bus

⚡ Apache Kafka 3.7 — KRaft Mode (no Zookeeper)  ·  Internal: :9092  ·  Controller: :9094  ·  auto.create.topics.enable=true
